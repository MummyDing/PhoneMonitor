# PhoneMonitor 手机监控器
An app used for monitor another cell phone
手机监控器需要两个手机实现操控，一个手机作为主控端，一个手机作为被控端，主控端可以通过该应用控制被控端。
##主要功能
主控端通过发送短信指令控制被控端，让被控端拨打电话，手机响铃，手机震动，手机定位。
